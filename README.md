本程序旨在适配电子墨水屏阅读器中关于编码转换和目标小说下载的功能

本程序同时发布手机版和电脑版，更新公告已内置

BUG提交和修复为“陌生人数码”售后群内的使用反馈

当前最新版本：2.3.1目前双端支持功能：
            
            1.小说TXT格式编码一键转换
            
            2.番茄小说网页链接一键下载对应小说
            
            3.增加对任意小说URL的解析下载功能

            4.增加内置的小说搜索引擎

            5.增加下载目录查看

            6.对于所有下载的小说，自动转为utf-8编码以适配水墨屏阅读器

使用方法：下载设备操作系统对应的软件后直接进入，程序内有对应的直观指导
